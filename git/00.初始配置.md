### 指定用户名和Email地址
````
git config --global user.name  "Github用户名"
git config --global user.email "Github注册邮箱"
````

```
### 生成SSH key
```
$ ssh-keygen -t rsa
$ cat ~/.ssh/id_rsa.pub　＃ copy file content 
$ ssh -T git@github.com  # check config 
